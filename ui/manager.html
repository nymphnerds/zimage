Installed module controls for the local Z-Image Turbo backend.

// MODEL WEIGHTS

Fetch the base model and the selected Nunchaku quantized weight into the module cache.

  • These are Nunchaku generation weights for Z-Image Turbo, not LoRA training precision.
  • Published weights: INT4 r32/r128/r256 and FP4 r32/r128.
  • LoRA training uses BF16 separately.
  • On 12-16 GB VRAM, INT4 r32 is the safer first choice.
  • Auto only supports r32/r128 because r256 is INT4-only.
